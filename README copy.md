# EPG App

React EPG (Electronic Program Guide) application built with Vite, TypeScript, and TanStack Router.

## Tech Stack

- React 19
- TypeScript
- Vite
- TanStack Router
- TanStack Query
- SCSS
- Zod
- Vitest
- ESLint
- Prettier

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
